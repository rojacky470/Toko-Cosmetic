Judul & deskripsi singkat:

“Jakicosmetic – Website booking & penjualan kosmetik (Laravel 11 + React + Filament)”.

Tech stack:
laravel sebagai content Manajement sistem

Backend: Laravel 11, Filament 4, MySQL.

Frontend: React, TypeScript, Vite, Tailwind.

Fitur utama:

Booking cosmetik, cart & checkout.

Bukti transfer (upload dan tampil di admin).

Additional photos untuk produk.

Halaman booking detail, testimoni, payment, index, booking finished, My-Booking detail, Category product, My-orders, My-cart Page.

Struktur folder:
/backend-laravel  → API, admin (Filament)
/frontend-react   → UI customer (Vite + React)
